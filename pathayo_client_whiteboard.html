Browse catalogue + upload reference

Each ticked sub-category becomes its own independent provider selection

STEP 05
Pick service providers
One tab per selected sub-category. Customer browses providers under each tab and selects one per sub-category. Provider cards show real-time availability on the event date.
Providers shown are filtered by availability on the customer's event date
Sort options: top rated, price low–high, most booked, nearest
Tapping a card opens full profile: gallery, reviews, pricing breakdown, availability calendar
Reference images from Step 3 are shared with selected provider after booking
"Request quote" option available for custom requirements outside standard pricing

STEP 08
Track event + rate providers
After booking, customer sees a live progress tracker. Each milestone is updated by the provider. Built-in chat allows communication without sharing personal contacts.
Status milestones: Confirmed → Provider accepted → Setup in progress → Completed
Provider updates status from their own dashboard
In-app chat — no personal number sharing required
After event completion, customer is prompted to rate each provider separately
Payment released from escrow after customer marks event complete
